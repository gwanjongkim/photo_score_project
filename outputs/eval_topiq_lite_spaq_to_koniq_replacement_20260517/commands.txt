# TOPIQ-lite SPAQ to KonIQ Fine-tuning Commands

The following command categories were used to create and evaluate the unified replacement candidate:

1. KonIQ Dataset Splitting:
   python -c 'import pandas as pd; from sklearn.model_selection import train_test_split; ...'
   - Created 80/10/10 train, val, test splits in data/processed/koniq10k/splits/.

2. Fine-tuning from SPAQ Weights:
   PYTHONPATH=. ./.venv_gpu/bin/python src/train/train_topiq_lite.py \
     --train_csv data/processed/koniq10k/splits/train.csv \
     --val_csv data/processed/koniq10k/splits/val.csv \
     --target_col normalized_mos --batch_size 8 --epochs 30 --lr 3e-5 \
     --freeze_backbone \
     --init_weights outputs/full_topiq_lite_spaq_frozen_e30_20260517/best.weights.h5 \
     --out_dir outputs/full_topiq_lite_spaq_to_koniq_frozen_e30_20260517

3. Comparative Evaluation (Replacement Track):
   - Benchmarked unified candidate against mobile_koniq and mobile_flive.
   - Evaluated on KonIQ test, SPAQ test, and FLIVE 1024 val.
   - Performed parity check with TFLite FP16 interpreter.

4. TFLite Export:
   - Builtin-only conversion (no Flex delegate).
   - FP32 and FP16 versions.
